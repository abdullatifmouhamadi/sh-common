# installing wordpress

```

wget https://github.com/WordPress/WordPress/archive/5.5.zip
unzip 5.5.zip

sudo cp -a WordPress*/. /srv/http/wordpress/








sudo chown -R http:http /srv/http/wordpress
sudo chmod -R 755 /srv/http/wordpress



```
